# MCP 境界

MCP サーバーは、エージェントアクセスのポリシー適用層です。

`mcp/server.py` が公開するツールは、mem0 に問い合わせる前に読み取り可能テナントの
絞り込みを適用します。書き込み操作は、設定された 1 つの書き込み先テナントを使います。

## ツール

- `search_memory`: 読み取り可能テナントを検索します。
- `remember`: 設定された書き込み先テナントに書き込みます。
- `related_repo_context`: リポジトリメタデータを考慮して検索します。
- `recent_project_memories`: プロジェクトに紐づく文脈を取得します。

## 読み取りと書き込みの分離

読み取り可能テナントと書き込み先テナントは `mem0.policy.yml` に設定します。

```yaml
read:
  - mimr-tech

write:
  - mimr-tech
```

書き込み先テナントは、自動的に読み取り可能テナントにも含めます。
これにより、同じエージェント境界で新しく書いた文脈を検索できます。
